This is an SDK to help develop applications that interact with EBU LIST.

# Usage

ebu-list-sdk uses [lerna](https://github.com/lerna/lerna) to manage multiple npm packages in the same repository.

- Bootstrap:

```
> npm i
> npx lerna bootstrap
```

- Build:
```
> npx lerna run build
```

- Run the demos:

```
> cd demos/<demo>
> npm start
```
